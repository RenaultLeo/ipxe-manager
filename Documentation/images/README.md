# Dossier des captures d’écran

Placez ici les images référencées dans la documentation.

## Convention de nommage

```
{numero-chapitre}-{section}-{detail}.png
```

Exemples :

- `01-login-page-complete.png`
- `04-isos-upload-form-filled.png`
- `08-menus-generated-tab-menu-ipxe.png`

## Conseils de prise de vue

- Résolution : **1920×1080** ou plus, thème sombre (défaut de l’UI).
- Masquer ou flouter les **mots de passe**, IP internes sensibles si la doc est publique.
- Utiliser la **même langue** que le chapitre (FR recommandé pour la doc interne).
- Un **curseur** ou **surbrillance** (outil OS) sur le bouton décrit aide la lecture.
- Pour les listes longues : capture **entière** + capture **zoom** sur la zone utile.

## Sous-dossiers (optionnel)

Vous pouvez organiser ainsi :

```
images/
  01-connexion/
  03-dashboard/
  04-isos/
  ...
```

Dans ce cas, adaptez le chemin dans le markdown du chapitre correspondant.
